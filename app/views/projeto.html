Projeto

O projeto de TCC se chama InCasa e é um sistema de automação residencial através de micro controlador Arduino utilizando API de reconhecimento de voz, seu público alvo são pessoas com mobilidade reduzida, idosos e deficientes fisicos com mobilidade reduzida.

O Projeto Incasa é dividido em 4 frentes:

  • Servidor(PHP)
  • Aplicativo(Android)
  • Arduino
  • Página Web

A partir do aplicativo android o usuario pode fazer comando de voz que são reconhecidos com a API Speech Text do Google e enviados para o servidor backend.

O servidor backend troca informações com o arduino enviando comandos e recebendo valores dos sensores.

Assim o aplicativo recebe as informações armazenadas no banco e exibe para o usuário.

O projeto também conta com uma página web, para gerenciamento do sistema via navegador

Funcionalidades

  • Comando de voz
  • Ligação de equipamentos eletro/eletrônicos
  • Controle de sensores
